=================================================
ボタン配置ブロック
=================================================
:doc:`button_block` は、各種の :doc:`button_submit`
を配置するためのブロックを定義するウィジェットである。
ブロック内の各ボタンは、 **UI標準** に沿って配置される。


コードサンプル
==================================

**実装成果物(サーバ動作)**

.. code-block:: jsp

    <button:block>
      <n:forInputPage>
        <button:check uri="./確認画面.jsp">
        </button:check>
      </n:forInputPage>
      <n:forConfirmationPage>
        <button:cancel uri="./登録画面.jsp">
        </button:cancel>
      </n:forConfirmationPage>
    </button:block>


仕様
=============================================
このウィジェットは各種ボタンの配置を制御するためのラッパーDIVを定め、
その中にボディ部に記述された内容をそのまま出力する。

**属性値一覧** 
  なし
