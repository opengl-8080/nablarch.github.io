=================================================
リストビルダー入力項目ウィジェット
=================================================
:doc:`field_listbuilder` は **UI標準 UI部品 リストビルダー**
の内容に準拠したリストビルダーのリストを出力する。


コードサンプル
==================================

**設計成果物(ローカル動作)**

  .. code-block:: jsp

    <field:listbuilder
        title="認可チェック単位"
        sample="認可チェック単位１|[認可チェック単位２]">
    </field:listbuilder>


**実装成果物(サーバ動作)**

  .. code-block:: jsp

    <field:listbuilder
      title="認可チェック単位"
      name="formdata.permissionUnits"
      id="permissionUnit"
      listName="allPermissionUnit"
      elementLabelProperty="permissionUnitName"
      elementValueProperty="permissionUnitId">
    </field:listbuilder>


仕様
=============================================

このウィジェットは :doc:`field_base` を用いて実装している。
:doc:`field_base` が実装する共通仕様についてはここでは記述しない。

**ローカル動作時の挙動**
  入力画面では **sample** に指定した項目を選択候補とするリストビルダーを表示する。
  **"[]"** で囲われた項目は、既に選択済みの項目として表示される。

  確認画面では、選択済み項目(**"[]"** で囲われた項目) のラベルの一覧を表示する。

**属性値一覧**  [**◎** 必須属性 **○** 任意属性 **×** 無効(指定しても効果なし)]

(:doc:`field_base` の共通属性は省略)
