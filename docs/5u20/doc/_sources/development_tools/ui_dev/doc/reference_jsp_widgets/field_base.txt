=================================================
入力項目ウィジェット共通テンプレート
=================================================
:doc:`field_base` は :doc:`field_text` や :doc:`field_checkbox`
などの入力項目UI部品に実装の実装で使用するテンプレートである。
このテンプレートでは以下の内容を実装している。

  - 各入力項目のレイアウト(ラベル、入力項目、項目別エラー、補助テキストなどの表示制御)
  - 必須項目の表現


コードサンプル
==================================
:doc:`field_base` は :doc:`../internals/jsp_widgets` の実装に用いるテンプレートなので、
業務画面JSPから直接使用することはない。
