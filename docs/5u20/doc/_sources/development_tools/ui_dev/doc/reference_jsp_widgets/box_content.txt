=================================================
コンテンツ用表示領域ウィジェット
=================================================
:doc:`box_content` は、コンテンツの表示領域を定義するウィジェットである。
各PJのポリシーに沿った表示領域を出力する。

コードサンプル
==================================

**実装成果物(サーバ動作)**

.. code-block:: jsp

      <box:content>
        ここに記載された説明文などは定義情報を持ったブロック要素として出力され、
        PJ共通のスタイル定義を設定できる。
      </box:content>

仕様
=============================================
定義情報を持ったブロックで囲んだボディ部の内容を出力する。
なおローカル動作とサーバ動作の挙動は同じである。

**属性値一覧**  [**◎** 必須属性 **○** 任意属性 **×** 無効(指定しても効果なし)]

==================== ============================== ============== ========== ========= 
名称                 内容                           タイプ         サーバ     ローカル  
==================== ============================== ============== ========== ========= 
cssClass             定義領域の\                    文字列         ○          ○
                     class属性を指定する。
