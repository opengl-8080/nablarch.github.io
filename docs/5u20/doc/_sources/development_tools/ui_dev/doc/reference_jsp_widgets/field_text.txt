=================================================
単行テキスト入力項目ウィジェット
=================================================
:doc:`field_text` は **UI標準 UI部品 単行テキスト入力**
の内容に準拠したテキストボックスを出力する。


コードサンプル
==================================

**設計成果物(ローカル動作)**

  .. code-block:: jsp

    <field:text title="漢字氏名"
                hint="全角50文字以内で入力してください。"
                required="true"
                maxlength="50">
    </field:text>


**実装成果物(サーバ動作)**

  .. code-block:: jsp

    <field:text title="漢字氏名"
                required="true"
                maxlength="50"
                hint="全角50文字以内"
                name="W11AC02.users.kanjiName">
    </field:text>


仕様
=============================================

このウィジェットは :doc:`field_base` を用いて実装している。
:doc:`field_base` が実装する共通仕様についてはここでは記述しない。

**ローカル動作時の挙動**
  入力画面では **sample** に指定した文字列を初期表示するテキストボックスを表示する。

  確認画面では、 **sample** に指定した文字列をラベル表示する。

**属性値一覧**  [**◎** 必須属性 **○** 任意属性 **×** 無効(指定しても効果なし)]

(:doc:`field_base` の共通属性は省略)
