===========================
関連文書
===========================

**UI標準**
  本機能が提供する各UI部品は、全てUI標準の内容に沿って設計・実装されている。
  このため、同書の内容を把握しておくことは、本書の理解のためには欠かせないものとなっている。

**JSP/HTML作成ガイド**
  各画面機能の設計者・開発者が、UI開発基盤を使用して業務画面JSPを作成する際に参照するガイド。
  画面作成作業の手順を理解しておくため、本書を読む前に必ず通読しておくことが望ましい。

**Nablarchアーキテクチャ解説書**
  業務画面の各JSPや、共通部品として提供されるタグファイル群は
  Nablarch Application Framework (NAF) が提供するタグライブラリなど、
  NAFの機能に依存したものが多く存在するため、同書のウェブアプリケーション基盤に関する部分を
  理解しておくことが望ましい。
