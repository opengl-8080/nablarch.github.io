How to Use the Log Aggregation Sample
=======================================

.. important::

  This sample uses a Nablarch 1.4 compliant API.

  When combining with versions later than Nablarch 1.4 series, customize as necessary.


This document describes how to use the sample implementation [#statistics]_  of log aggregation.

.. [#statistics]

  The log aggregation function aggregates the access status and processing time, etc. based on the logs and database status output by application in the production environment.
