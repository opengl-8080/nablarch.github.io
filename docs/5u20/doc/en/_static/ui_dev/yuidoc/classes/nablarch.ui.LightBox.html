nablarch.ui.LightBox Class

Defined in: nablarch-dev-ui_tool-base-core\ui_local\js\nablarch\ui\LightBox.js:6
Module: nablarch.ui

Available since 1.3

モーダルダイアログ

モーダルダイアログを表示する。

マークアップ仕様

ダイアログ内に表示するDOM要素に対して、マーカーCSSクラス nablarch_LightBox を指定する。

<div class="nablarch_LightBox
            -open  '#contextMenu li.widgetList click'
            -close 'button.close click'">
  <h2>ウィジェット一覧</h2>
  <div id="widgetListView"></div>
  <button class="close">ウィジェット一覧を閉じる</button>
</div>
